# 🌍 COVID-19 Global Analytics Dashboard

An interactive dashboard built with **Python**, **Streamlit**, and **Plotly Express** to visualize global COVID-19 trends, mortality rates, and recovery statistics.

## 🚀 Live Demo
[View the Dashboard on Streamlit Cloud](https://coviddashboardpy.streamlit.app/)

## 📊 Project Overview
This project processes a global COVID-19 dataset to provide actionable insights. It cleans raw data by removing unnecessary geographical coordinates and focuses on key performance indicators (KPIs) like total cases, deaths, and active case percentages.

### Key Features
* **Top 10 Analysis:** Vertical and horizontal bar charts for Cases, Deaths, and Recoveries.
* **Death Rate Calculation:** A custom metric comparing deaths to total cases per country.
* **Interactive Visuals:** Hover-over details and zooming capabilities provided by Plotly.
* **Dynamic Layout:** Responsive design using Streamlit columns for a professional UI.
