# 🌍 COVID-19 Global Analytics Dashboard

An interactive web-based dashboard built with **Python**, **Streamlit**, and **Plotly** to visualize real-time global COVID-19 data. 

### 🚀 [Live Dashboard Link](https://coviddashboardd.streamlit.app/)

---

## 📊 Project Overview
This project provides a comprehensive look at the pandemic's impact across different countries. By utilizing data from the `disease.sh` API, the dashboard offers insights into total cases, mortality rates, recovery statistics, and the relationship between active and critical cases.

### Key Features:
* **Top 10 Analysis:** Visualize the countries most affected by total cases, deaths, and recoveries.
* **Death Rate Insights:** Comparative analysis of death rates per country.
* **Interactive Visuals:** Dynamic charts that allow for zooming and data inspection.
* **Cleaned Data View:** Ability to toggle and view the processed dataset directly.
